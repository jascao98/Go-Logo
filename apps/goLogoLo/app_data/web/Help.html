Help

The goLogoLo Editor is a desktop Java application for creating and editing your own brand of logos.

Step 1: Create new logo

To start, click on create new logo in the welcome menu to get started, or load in a logo file from previous uses.

Step 2: Edit

Click on the logo Components to get started with the editing usage of this application.

Step 3: Try!

The application has many features for how you want to customize your logos. On the left side, there holds logo information in a chart that the user can move up, down, and edit. In the middle is the canvas where all the magic happens. It is where all the editing takes place. On the right side, you can add in logo components, and change the font, gradient and color of the logo components. Finally, on the top of the application holds the standard toolbar which hold functions such as save,export,zoom in and etc..

Step 4: Save the Logo

Once you've completed your work, you can press the save button and give your file a name, which does not necessarily have to be the same as the name you specified in the name text field.

Step 5: Export your Logo

Press the Export button when you wish to export your completed logo.
